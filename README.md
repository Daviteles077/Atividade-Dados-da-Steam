## 📌 Sobre o Projeto

Este repositório contém a atividade prática de **Ciência de Dados** desenvolvida pelo **Grupo 6**. O objetivo principal é explorar um conjunto de dados reais da **Steam**, levantar hipóteses, criar visualizações objetivas e extrair *insights* valiosos sobre o mercado de jogos.

> 💡 **Premissa de Ouro:** *"O foco não é apenas escrever código, mas principalmente interpretar os dados, escolher visualizações adequadas e contar uma história através deles."*

---

## 🚀 Diretrizes da Atividade

- [x] **Tempo de Apresentação:** Rigorosamente **5 minutos** via Notebook `.ipynb`.
- [x] **Contextualização:** Breve introdução sobre a natureza dos dados analisados.
- [x] **Regra dos 6:** Responder a pelo menos **6 perguntas** norteadoras utilizando no mínimo **6 gráficos**.
- [x] **Abordagem Clean:** Visualizações seguindo a lógica do *"menos é mais"*, simples e diretas.
- [x] **Uso Ético de IA:** Inteligência Artificial utilizada para otimizar o código de plotagem, mantendo o entendimento total de cada etapa pela equipe.

## 📊 Perguntas Norteadoras & Insights Esperados

| # | Pergunta Analisada | Tipo de Gráfico | Insight Alvo |
|---|--------------------|-----------------|--------------|
| **1** | Como o volume de lançamentos cresceu ao longo dos anos? | `plt.plot` (Linha) | Identificar o "boom" do mercado indie. |
| **2** | Qual a proporção de jogos Free-to-Play vs. Pagos? | `plt.pie` (Rosca) | Compreender a predominância de modelos de monetização. |
| **3** | Qual a média de preço dos jogos por grandes gêneros? | `plt.barh` (Horizontal) | Descobrir quais categorias cobram mais caro por seus títulos. |
| **4** | Jogos mais caros possuem melhores avaliações dos usuários? | `plt.scatter` (Dispersão) | Correlacionar investimento financeiro com satisfação. |
| **5** | Quais são os 5 gêneros mais frequentes na plataforma? | `plt.bar` (Vertical) | Mapear a saturação de nichos de mercado. |
| **6** | Modo Multiplayer impacta no volume de avaliações positivas? | `plt.bar` (Comparativo) | Avaliar se o fator social impulsiona o engajamento. |
